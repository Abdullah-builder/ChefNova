# 🍳 ChefNova

IMG_0036.jpeg

> AI-powered cooking assistant to discover recipes, get smart ideas, and cook like a pro.

---

## 🧠 Features

- 🔍 **Find Recipes** — Discover meals instantly  
- 💡 **Get Ideas** — Missing ingredients? No problem  
- 🤖 **AI Suggestions** — Improve taste, texture, and creativity  
- ⭐ **Favorites** — Save recipes you love  

---

## 📱 How to Use

1. Tap **Find Recipes** to explore meals
2. Tap **Get Ideas** if you're missing ingredients
3. Read **AI Suggestions** for expert tips
4. Tap **sign in** before saving your favorites
